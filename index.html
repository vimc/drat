VIMC drat repository

VIMC internal packages that are not yet (and in some cases ever) going on CRAN

Packages

  • montagu (v 0.1.0, ): Interface with 'montagu'
  • orderly (v 0.1.1, ): R support for montagu-reports
  • vaultr (v 0.0.1, ): API Client for Vault

If drat is not installed, first install from CRAN with

install.packages("drat")

To install packages, run

drat:::add("vimc")

after which packages can be installed as usual, for example:

install.packages("montagu")
